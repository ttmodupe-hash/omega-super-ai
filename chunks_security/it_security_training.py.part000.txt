- IDOR (Insecure Direct Object References) — manipulating IDs in URLs

```python
async def get_invoice(invoice_id: str, user: User = Depends(get_current_user)):
    invoice = db.invoices.find_one({"_id": invoice_id})
    if not invoice or invoice["owner_id"] != user.id:

    return invoice
```

**Most critical web vulnerability.** Occurs when untrusted data is sent to an interpreter as part of a command or query.

**Types of SQLi:**

```python
os.system(f"nslookup {user_input}")
```

```python
subprocess.run(["nslookup", user_input], capture_output=True)
```
